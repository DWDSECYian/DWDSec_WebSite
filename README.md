# About DWDSec Infomation

This is DWDSEC Network Security Laboratory

安装流程:

```bash
//1.安装node.js
//2.安装yarn

npm install --global yarn

//3.安装vuepress

yarn add -D vuepress@next

//4.运行官网
yarn docs:dev

```

目录结构说明
```

- docs
    -- Member       //成员介绍
    -- wp           //实验室文章
    -- object       //实验室所开设的项目
    -- .vuepress    //vuepress资源配置等文件

```

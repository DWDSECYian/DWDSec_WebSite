  - title: MISC
    details: MISC 即 Miscellaneous 的缩写，意为杂项，涉及信息搜集、取证、编解码、加解密、隐写、图片处理、压缩包、编程等
  - title: WEB
    details: WEB 涉及操作系统和 Web 系统安全，包括 SQL 注入、XSS 跨站脚本、CSRF 跨站请求伪造、文件上传、文件包含、框架安全、PHP 常见漏洞、代码审计等
  - title: CRYPTO
    details: CRYPTO 指密码学，一般是要想办法识别出密码算法，然后进行暴力破解，或者利用密码体制的漏洞进行破解
  - title: BINARY
    details: BINARY 指二进制安全，主要包括 REVERSE 和 PWN
  - title: REVERSE
    details: REVERSE 指软件代码逆向，主要指对软件的结构，流程，算法，代码等进行逆向拆解和分析，主要应用于软件维护，软件破解，漏洞挖掘，恶意代码分析
  - title: PWN
    details: PWN = 理解目标工作原理 + 漏洞挖掘 + 漏洞利用，指的是通过通过程序本身的漏洞，编写利用脚本破解程序拿到主机的权限
